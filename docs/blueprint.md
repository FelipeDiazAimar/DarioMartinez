# **App Name**: Darío Martínez Computación Web

## Core Features:

- Service Listing: Display the list of computer services: PC/notebook repair, Windows/software installation, hardware upgrade, preventative maintenance, networking and technical support for home and business.
- WhatsApp Quick Contact: A button to initiate a WhatsApp chat for quick customer contact.
- Contact Form: A simple contact form for inquiries.
- About Section: A brief description of the business emphasizing trust, experience, and personalized service.
- Basic SEO: Implement basic SEO optimization for local searches.

## Style Guidelines:

- Primary color: Deep blue (#00308F) to reflect trust and professionalism, based on the logo.
- Background color: Light blue (#E0F2F7), a heavily desaturated version of the primary, for a clean, modern background.
- Accent color: A slightly violet-blue (#3050AF) that provides enough saturation and brightness to stand out from the primary, drawing attention to calls to action.
- Body and headline font: 'Inter', a sans-serif font, to maintain a modern and clear aesthetic, suitable for both headlines and body text.
- Use modern, outline-style icons for each service offered to enhance visual appeal and clarity.
- Clean, responsive layout optimized for both desktop and mobile viewing. Single-page design for simplicity or multi-page for detailed content.
- Subtle animations for smooth transitions and engagement without being distracting.
